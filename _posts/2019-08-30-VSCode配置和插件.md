# 配置VSCode舒心的写Python代码

### 如何配置VSCode



### 插件

> 工欲善其事必先利其器，开发前先把所以提升效率的利器搭好会让今后慢慢的编程长路舒服很多

- `Settings Sync` 这个插件可以实现同步你的vscode设置，包括setting文件，插件设置等

- `Python`  让VS Code支持python编程,提供了代码分析，高亮，规范化等很多基本功能

- `Material Theme`  VSCode主题（多种）

- `vscode-icons`  改变vscode原有的icon(颜控)

- `filesize`  一款在左下角显示文件大小的插件

- `Python Indent` 在每次换行的时候会解决Python Indentation的一些问题

- `Code Runner` | `Output Colorizer` 

    - 右键里有在终端中直接运行Python文件，并对输出给与颜色标注，方便观看

- `vscode-fileheader`

    - 使用该插件可以给自己的文件添加文件头，这样更方便维护自己的代码的权益

- Markdown 相关插件

    - `evermonkey`  Markdown support for Evernote.
    
    - `markdown`  Markdown 编辑器 

    - `Markdown All in One` & `Markdown Preview Enhanced`
    
        - 让Markdown更好的在vscode进行编辑和预览，查看这两个组件的详情就知道了，非常简洁明了。

    - `writing4cn`  Markdown中文格式化方案

        - 打开控制面板 ctrl+shift+p 
        - 输入 pangu 确定

- `Chinese (Simplified) Language Pack for Visual Studio Code` 适用于 VS Code 的中文（简体）语言包

- `Excel Viewer`  Excel Viewer

- `autoDocstring`  自动的把所有输出输出参数以一定的风格放到函数或者类注释里面，实乃神器 
    
- Press enter after opening docstring with triple quotes (""" or ''')
    
- `Github History` 这个组件会以GUI的格式展示Git的历史commit信息，更加直观方便

- `Python Preview`

    - 展示Python中内置变量的值，方便有时候调试展示值。

- 代码错误提示 linting

    - `flake8`  代码错误提示
      
        - `pip install -U flake8 --user`
        - 打开settings.json，进行配置
        ```
            "python.linting.flake8Enabled": true,
            "python.linting.enabled": true,
            "files.autoSave": "afterDelay",
            "files.autoSaveDelay": 500,
        ```

- 代码自动格式化

    - `black for autoFormatting`    让一坨难看的shit代码变成好看的shit代码。。。最强的是你不用每次手动去formatting，只需要在配置文件里面一键设置**每次保存代码的时候去formatting**，效率那是大大的快！

        - 安装Python代码格式化工具Black：`pip install black`
            
            - `black xxx.py`
        - 在vscode中打开搜索栏，输入settings，点击第一行 Preferences...
        - 进入配置文件添加红框中的相应配置
            - 打开settings.json，进行配置
            ```
                "autoDocstring.docstringFormat": "numpy",
                "editor.formatOnSave": true,
                "python.formatting.provider": "black",
                "python.formatting.blackArgs": [
                    "--line-length",
                    "120"
                ],
            ```

    - `yapf`  自动格式化代码

        - 安装yapf之后在VScode中按 `Alt+Shift+F` 即可自动格式化代码

- 代码补全工具

    - `aiXcoder` 北大出品（不太好用）
        - 一键输入整行代码，敲代码不存在的
        - 需上传代码
    - `kite`    实时补全代码，Python 之父发声力挺
        - 代码补全更智能、高效
        - 只支持 python 语言
        - kite是一款安装包+插件的工具，首先需要到官网下载kite的安装包，安装作为引擎，安装之后打开相应的编辑器或IDE安装kite的插件，然后就可以使用了，不用像sublime、vscode那样需要安装一堆插件还要到设置中配置Python路径之类的。
    - `TabNine` 「程序员的杀手级应用」
        - 功能强大，支持度广泛
        - cpu 消耗大，笨重
