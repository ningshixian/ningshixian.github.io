>随便整理的一些好用的python三方库


# python三方库

|python库|描述|
|:----:|----|
|Chardet|字符编码探测器，可以自动检测文本、网页、xml的编码。|
|colorama|主要用来给文本添加各种颜色，并且非常简单易用。|
|difflib|[Python]标准库，计算文本差异|
|Levenshtein|快速计算字符串相似度|
|fuzzywuzzy|字符串模糊匹配|
|esmre|正则表达式的加速器|
|xlwt/xlrd|读写Excel格式的数据文件|
|BeautifulSoup|基于Python的HTML/XML解析器，简单易用, 功能很强大,即使是有bug，有问题的html代码，也可以解析|
|lxml|快速，易用、灵活的HTML和XML处理库，功能超强，在遇到有缺陷、不规范的xml时，Python自带的xml处理器可能无法解析。报错时，程序会尝试再用lxml的修复模式解析。|

## 大数据与科学计算

|python库|描述|
|:----:|----|
|Pandas|Pandas，python实现的类似R语言的数据统计、分析平台|
|SciPy|SciPy，开源的Python算法库和数学工具包，SciPy包含的模块有最优化、线性代数、积分、插值、特殊函数、快速傅里叶变换、信号处理和图像处理、常微分方程求解和其他科学与工程中常用的计算。|
|orange|orange，橙色，数据挖掘，数据可视化，通过可视化编程或Python脚本学习机分析。|
|Biopython|Biopython，免费的生物计算工具包|
|NLTK|NLTK（natural language toolkit)，是python的自然语言处理工具包|
|textblob|textblob，提供API为自然语言处理、分解NLP任务。基于NLTK和Pattern模块。|
|jieba|jieba，结巴，中文分词工具。|
|snownlp|snownlp，用于处理中文文本库。|
|scikits.learn|scikits.learn，构建在SciPy之上用于机器学习的 Python 模块。它包括简单而高效的工具，可用于数据挖掘和数据分析。涵盖分类，回归和聚类算法，例如SVM， 逻辑回归，朴素贝叶斯，随机森林，k-means等算法|
|Joblib|Joblib,Python提供的轻量级的流水线工具函数。|
|PyMySQL|PyMySQL,纯Python写的 MySQL驱动程序，兼容mysql-python|
|Requests|Requests，用Python语言编写，基于 urllib的开源 HTTP 库。它比 urllib 更加方便，更加 Pythoner。|

## Web

|python库|描述|
|:----:|----|
|you-get|you-get，Python3写的视频下载工具，可用于YouTube/Youku优酷/Niconico视频下载Coursera，从coursera.org下载视频，可重新命名文件wikiteam，wiki下载工具。|

|Tkinter|Tkinter，是Python事实上的标准GUI软件包。|
|PyQt|用于Python的QT开发库|

|Py2exe|python脚本转换为windows上可以独立运行的可执行程序|
|pyinstaller|-转换成独立的可执行文件的Python程序（跨平台）。|
|glue|胶胶，一个简单的命令行工具，生成CSS Sprites。|
|pytest|成熟的全功能的Python测试工具|
