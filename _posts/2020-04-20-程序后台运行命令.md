# 后台跑程序 nohup&screen

1. 第一种方法：nohup

   ```
   nohup python -u flask_test.py > log.txt 2>&1 &
   ```

   为避免python的输出缓冲，将程序中print的内容写入日志，使用-u参数，使得python不启用缓冲.

   

2. 第二种方法：screen

   ```
   创建窗口
   screen -S intent
   python xxx.py
   
   查看哪些窗口正在运行
   screen -ls
   
   恢复会话窗口
   screen -r 线程ID
   
   会话分离
   ctrl+a 然后 d
   
   杀死会话窗口
   kill -9 线程号
   screen -wipe
   ```
