## 问题3

让我们定义两个简单的函数：

```
def f():
    l = [1]
    def inner(x):
        l.append(x)
        return l
    return inner
    
def g():
    y = 1
    def inner(x):
        y += x
        return y
    return inner
```

以下命令将产生什么结果？

```
>>f_inner = f()
>>print(f_inner(2))

>>g_inner = g()
>>print(g_inner(2))
```
